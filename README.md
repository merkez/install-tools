> **ALTHOUGH IT IS TESTED ON UBUNTU 20, USE IT AT YOUR RISK, CHECK THE SCRIPT BEFORE USING TO UPDATE VERSIONS AND OTHER STUFF THAT YOU MAY WANT TO UPDATE**

```bash

$ sudo bash install-tools.sh
```
